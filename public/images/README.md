# Service Block Images

Place your 6 service block images in this folder with these exact names:

1. `drain-unclog.jpg` - For "AED 199 Unclogs Drain" block
2. `kitchen-drains.jpg` - For "Kitchen Drains: Sink" block  
3. `bathroom-drains.jpg` - For "Bathroom Drains: Shower, Tub, Sink" block
4. `laundry-room.jpg` - For "Laundry Room: Main" block
5. `main-sewer-line.jpg` - For "Main Sewer Line" block
6. `accessible-cleanout.jpg` - For "Accessible Cleanout" block

**Supported formats:** `.jpg`, `.jpeg`, `.png`, `.webp`

Once you add the images here, they will automatically appear on the website!
